# ToonTanks
ToonTanks는 언리얼 엔진5를 이용한 간단한 아케이드 슈터 게임입니다.

# How to Play
- Build/Windows의 ToonTanks.exe를 통해 실행할 수 있습니다.
- 공격 : 마우스 좌클릭
- 앞, 뒤 이동 : W, S
- 좌, 우 회전 : A, D
- 전방으로 대쉬 : Shift
- 점프 : Space
- 재시작 : R
- 게임 종료 : ESC
